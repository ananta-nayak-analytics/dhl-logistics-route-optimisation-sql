# DHL Logistics Route Optimization (SQL Project)

## 📌 Overview
This project focuses on optimizing delivery routes using SQL to improve logistics efficiency and reduce operational delays in a DHL-based dataset.

## 🛠 Tools Used
- MySQL

## 🔍 Key Tasks Performed
- Data cleaning (handling duplicates and inconsistent date formats)
- Writing optimized SQL queries
- Analyzing delivery performance and route efficiency

## 📊 Key Insights
- Identified inefficient delivery routes affecting performance
- Improved route planning strategies
- Enhanced overall logistics efficiency

## 📁 Files Included
- DHL_script.sql → Contains all SQL queries used in the project  
- SQL Project.pptx → Detailed project explanation and insights  

## 🎯 Outcome
This project demonstrates how SQL can be used to analyze logistics data and generate actionable insights for better decision-making.
